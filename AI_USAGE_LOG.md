## 🧠 Reflection

AI significantly accelerated initial development by generating scaffolding and suggesting feature ideas such as quiz feedback and analytics.

However, all generated code required manual review and validation. I ensured that:

* The logic matched application requirements
* The data flow between frontend and backend was correct
* The implementation aligned with the existing project structure

This process helped reinforce my understanding of full-stack integration and debugging.

---

## ⚠️ Limitations Encountered

* Some generated logic required modification before it worked correctly
* Certain outputs did not align with the current project structure
* Documentation suggestions needed verification against actual implementation

---

## 🎯 Key Takeaway

AI tools were used as **development accelerators**, not replacements for understanding.

The final implementation required:

* Careful debugging
* Manual integration
* Validation of functionality

This approach ensured both **correctness of the application** and **improvement in my technical understanding**.
